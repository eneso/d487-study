# D487 Security Study Guide

An interactive multiple choice quiz website for studying CompTIA Security+ (D487) certification content.

## Features

- **Interactive Quiz**: Answer 200 multiple choice questions across 10 lessons
- **Lesson Navigation**: Filter questions by lesson or view all
- **Progress Tracking**: Real-time score and completion percentage
- **Instant Feedback**: See correct answers immediately after submission
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **No Backend Required**: Runs entirely in the browser using vanilla JavaScript

## Quiz Content

### Lessons Covered
1. Why build security in (SDLC + SDL basics)
2. SDL programs and maturity models (BSIMM, OWASP, NIST, CVE)
3. SDLC approaches (Waterfall, V-model, Agile, Scrum, XP)
4. SDL Phase A1: Security Assessment + Requirements
5. SDL Phase A2: Architecture + Threat Modeling
6. SDL Phase A3: Design & Development testing foundations
7. SDL Phase A4: Code review + AppSec tooling
8. SDL Phase A5: Ship (final review + scanning + pen testing)
9. Post-release support + PSIRT + CVSS + M&A
10. Modern environments + OpenSAMM + BSIMM categories + STRIDE recap

### Question Features
- **200 total questions** - 20 questions per lesson
- **Multiple choice format** - 4 options per question
- **Instant grading** - See if you're correct immediately after submission
- **Performance tracking** - Track your overall score percentage

## How to Use

1. **Select a Lesson** - Click on lesson buttons to filter questions (or view all)
2. **Read the Question** - Each question includes the lesson number
3. **Choose Your Answer** - Click on an option to select it
4. **Navigate** - Use Previous/Next buttons to move between questions
5. **Submit Answers** - Click "Submit Answers" to see your score
6. **Review Results** - Correct answers are highlighted in green
7. **Reset** - Click "Reset All" to start over
